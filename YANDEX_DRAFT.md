# Тексты для черновика Яндекс Игр — «Кликер 67»

Используйте **одинаковое название «Кликер 67»** во всех полях черновика, на иконке, скриншотах и в игре.

## Название (RU / EN)

```
Кликер 67
```

```
Clicker 67
```

## Краткое описание (RU)

```
Кликер 67 — жми на число 67, покупай улучшения и открывай достижения. Соревнуйся в таблице лидеров по очкам.
```

## Полное описание (RU)

```
Кликер 67 — бесконечный кликер с прокачкой.

Жми на большую кнопку 67, зарабатывай очки и покупай улучшения: усиление клика, автодоход, множители. Открывай достижения, следи за прогрессом до 10 миллионов очков и соревнуйся в таблице лидеров.

Особенности:
• 12 видов улучшений с глубокой прокачкой
• Автодоход и множители
• Достижения и система престижа
• Таблица лидеров
• Сохранение прогресса
• Буст ×2 за просмотр рекламы
```

## Как играть (RU)

```
1. Нажимай на кнопку 67, чтобы зарабатывать очки.
2. Во вкладке «Улучш.» покупай усиления за очки.
3. Открывай достижения и следи за прогрессом.
4. Во вкладке «Лидеры» смотри рейтинг и вводи имя.
5. При 1 000 000+ очков доступен престиж для ещё большего дохода.
```

## Иконка

На иконке должно быть написано **«Кликер 67»** — так же, как в названии черновика и в самой игре.

## Таблица лидеров в консоли

Создайте таблицу лидеров с техническим именем:

```
score
```

Тип: числовой, сортировка по убыванию.
